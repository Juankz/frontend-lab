¿Que es la semántica?

La semántica en sí no contribuye a estilizar la página web, sino que crea una estructura sólida del documento web

Significado

Cuando hablamos de semántica, estamos hablando de significado. La semática me ayuda a definir que quiere decir un elemento determinado en HTML

Atributos

Los atributos son propiedades de un elemento, existen atributos globales (Ej: id ó class) y locales por ejemplo src en la etiqueta <img>

Elementos Semánticos

La mayoría de elementos HTML son semánticos. Algunos importantes son:

  • <header>

    Area para contenido introductorio, contiene información importante acerca del contenido de la página web, una introducción, logo y nombre del sitio, no tiene que ir siempre al principio

  • <nav>

    Contiene enlaces para poderse mover por el sitio. Un sitio puede tener varios elementos de navegación

  • <section>

    Como su nombre lo sugiere, delimita secciones de un documento, por ejemplo capítulos. Usado tipicamente en contenido que no tiene sentido por sí mismo

  • <article>

    Define contenido independiente que tiene sentido por sí mismo fuera del documento. Por ejemplo un blog post

  • <aside>

    Contenido adicional cuya función es ayudar al contenido principal pero no hace parte de este, por ejemplo una explicación pequeña sobre un término del texto principal

  • <footer>

    Contiene información del autor, copyright, información de contacto y en ocasiones un menú

  • <details>

    Contenido que puede ser mostrado u ocultado por el usuario

    Dentro de la etiqueta <summary> se coloca el texto que se verá siempre visible

  • <figure> y <figcaption>

    Une una imagen con una descripción de la imagen

  • <mark>

    Efectivamente sirve para Resaltar texto

  • <code>

Elementos no semánticos

Existen dos elementos no semánticos principales:

  • <div>

    Es un contenedor de un bloque de contenido

  • <span>

    Es un contenedor de una línea de contenido

    • Ambos casos se usan para facilitar el uso de estilos agrupando ciertos elementos o manipulación de elementos a través de javascript
